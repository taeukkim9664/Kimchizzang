빠른 시작 가이드

1

김프 설치하기

공식 웹사이트에서 운영체제에 맞는 버전을 다운로드하고 설치하세요.

# Ubuntu/Debian sudo apt install gimp
2

첫 번째 이미지 열기

파일 → 열기 메뉴에서 이미지를 선택하거나 드래그 앤 드롭으로 열 수 있습니다.

단축키: Ctrl+O
3

기본 편집 도구 사용

도구 상자에서 브러시, 선택, 텍스트 도구 등을 사용해 이미지를 편집하세요.

브러시: B, 선택: R, 텍스트: T
4

이미지 저장하기

편집이 완료되면 파일 → 내보내기 메뉴에서 원하는 형식으로 저장하세요.

단축키: Ctrl+Shift+E
